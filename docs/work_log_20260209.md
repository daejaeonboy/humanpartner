## 1. 이메일 발송 기능 개선

- **문제점**: 로컬 개발 환경(`localhost`)에서 회원가입 인증 이메일 발송 시 연결 거부 오류 발생.
- **원인**: Firebase Hosting의 Rewrite 규칙이 로컬 개발 서버에는 적용되지 않음.
- **해결**:
  - `src/utils/email.ts` 수정: 실행 환경이 `localhost`인지 감지하여, 로컬일 경우 클라우드 함수 URL(`https://...`)을 직접 호출하도록 분기 처리.
  - `vite.config.ts`: 불필요해진 Proxy 설정 제거 (서버 안정성 확보).
