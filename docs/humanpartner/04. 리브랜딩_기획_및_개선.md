# 리브랜딩 기획 및 개선 사항 (행사어때)

## 1. 리브랜딩 개요
*   **기존**: Mice Partner (마이스파트너) - 다소 딱딱하고 기업 중심적인 느낌
*   **변경**: **행사어때** - 친근하고 직관적인 네이밍, 사용자에게 제안하는 듯한 느낌
*   **브랜드 컬러**: `#FF5B60` (Vibrant Coral) - 젊고 활기찬 에너지, 행사의 즐거움을 표현

## 2. 주요 개선 사항
### 디자인/UX
*   **시각적 임팩트 강화**: 기존의 차분한 네이비 톤에서 눈에 띄는 코랄 컬러로 변경하여 사용자 주목도 향상.
*   **일관성 있는 경험**: 로고, 파비콘(추후 교체 필요할 수 있음), 텍스트, 브랜드 컬러를 통일하여 브랜드 정체성 확립.

### 시스템
*   **Tailwind 설정 최적화**: `tailwind.config.js`에서 브랜드 컬러를 중앙 관리하도록 설정하여, 추후 컬러 변경 시 유지보수 용이성 확보.
*   **템플릿화**: `cmsApi`나 `Hero` 컴포넌트 등에서 텍스트가 하드코딩된 부분을 찾아 교체함으로써, 향후 텍스트 변경이 용이하도록 구조 개선 (일부 하드코딩은 CMS로 관리되도록 전환 고려 필요).

## 3. 향후 발전 계획 (Planning)
*   **로고 가시성**: 헤더 배경색에 따라 로고(이미지)의 가시성을 체크하고, 필요 시 다크 모드용/화이트 로고 별도 운영 고려.
*   **파비콘 업데이트**: 현재 로고에 맞는 파비콘(`favicon.ico`) 제작 및 교체 필요.
*   **OG 태그/SEO 강화**: 변경된 브랜드 명에 맞춰 소셜 미디어 공유 시 노출되는 이미지와 문구를 지속적으로 모니터링하고 최적화.
*   **문구 톤앤매너**: 단순히 이름만 바꾸는 것을 넘어, 사이트 전반의 문구(Copywriting)를 '행사어때'라는 친근한 브랜드 톤에 맞춰 수정 ("제공합니다" → "함께해요" 등).

## 4. 에이전트 노트
*   브랜드 컬러 변경(`brand-500`)은 버튼, 강조 텍스트 등 주요 UI 요소에 즉시 반영됩니다.
*   기존 네이비 컬러가 사용되던 배경이나 텍스트 중 가독성이 떨어지는 부분이 있는지 육안 확인이 필요합니다.
