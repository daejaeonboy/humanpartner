# 🔍 검색엔진(네이버/구글) 사이트 등록 가이드

우리 웹사이트(`https://human-partner.web.app`)를 사람들이 검색했을 때 나오게 하려면, **반드시 직접 등록**을 해줘야 합니다. 천천히 따라해보세요!

---

## 🟢 1. 네이버 서치어드바이저 (Naver Search Advisor)
네이버 검색 결과에 노출시키기 위한 과정입니다.

### 1단계: 사이트 등록
1. [네이버 서치어드바이저](https://searchadvisor.naver.com/) 접속 및 로그인
2. 우측 상단 **[웹마스터 도구]** 클릭
3. **사이트 등록** 입력창에 `https://human-partner.web.app` 입력 후 엔터
4. **사이트 소유확인** 단계가 나옵니다.
   - **"HTML 태그"** 방식을 선택하세요.
   - `<meta name="naver-site-verification" content="..." />` 형태의 코드가 보입니다.
   - **[중요]** 이 코드를 복사해서 저에게 알려주세요! 제가 사이트에 넣어드려야 인증이 완료됩니다.

### 2단계: 사이트맵 제출 (제가 코드를 넣은 후 진행)
1. 소유확인이 완료되면 좌측 메뉴에서 **[요청]** > **[사이트맵 제출]** 클릭
2. 입력창에 `sitemap.xml` 입력 후 확인 버튼 클릭

---

## 🔵 2. 구글 서치 콘솔 (Google Search Console)
구글 검색 결과에 노출시키기 위한 과정입니다.

### 1단계: 속성 추가
1. [구글 서치 콘솔](https://search.google.com/search-console) 접속 및 로그인
2. 좌측 상단 **속성 검색** > **+ 속성 추가** 클릭
3. **URL 접두어 (오른쪽 박스)** 방식을 선택하세요.
4. `https://human-partner.web.app` 입력 후 [계속] 클릭
5. **소유권 확인** 단계가 나옵니다.
   - **"HTML 태그"** 항목을 열어보세요.
   - `<meta name="google-site-verification" content="..." />` 형태의 코드가 보입니다.
   - **[중요]** 이 코드도 복사해서 저에게 알려주세요!

### 2단계: 사이트맵 제출 (제출 후 진행)
1. 인증 완료 후 좌측 메뉴 **[Sitemaps]** 클릭
2. **새 사이트맵 추가** 란에 `sitemap.xml` 입력 후 [제출] 클릭

## 🚀 지금 해야 할 일 요약
1. 네이버/구글에 접속해서 **HTML 태그(메타 태그)** 2개를 발급받으세요.
2. 발급받은 **긴 코드들을 그대로 복사해서 저에게 채팅으로 보내주세요.**
3. 제가 코드를 사이트에 심고 배포하면, **"소유권 확인" 버튼**을 눌러 인증을 마무리하시면 됩니다.
4. 마지막으로 **사이트맵 제출**까지 하면 끝!
