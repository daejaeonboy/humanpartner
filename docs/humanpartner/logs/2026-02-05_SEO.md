# 2026년 2월 5일 (수) SEO 최적화 분석 및 적용 보고서

## 📊 1. SEO 현황 분석 및 조치 사항
사용자분의 요청에 따라 웹서비스의 검색엔진 최적화(SEO) 상태를 진단하고 기본적인 최적화 작업을 수행했습니다.

### ✅ 조치 완료 사항
| 구분 | 내용 | 적용 파일/위치 | 효과 |
|------|------|----------------|------|
| **검색 로봇 제어** | `robots.txt` 생성 | `/public/robots.txt` | 검색엔진에게 크롤링 허용/차단 범위를 알려줌 (관리자/마이페이지 차단) |
| **사이트맵** | `sitemap.xml` 생성 | `/public/sitemap.xml` | 검색엔진이 내 사이트의 페이지 구조를 쉽게 파악하게 함 |
| **동적 메타 태그** | `react-helmet-async` 설치 및 적용 | `index.tsx`, `MainPage.tsx`, `ProductDetail.tsx` | 페이지마다 다른 제목(Title)과 설명(Description)을 노출하여 클릭률 상승 유도 |

## 📘 2. 비개발자를 위한 SEO 핵심 가이드 (필독)
비개발자(기획자/마케터/운영자) 관점에서 알고 계셔야 할 내용을 정리해드립니다.

### 1) SPA(Single Page Application)의 한계와 대책
- **현재 상황**: 우리 서비스는 React로 만들어진 SPA입니다. (페이지 이동 시 깜빡임 없이 부드럽게 내용만 바뀜)
- **문제점**: 네이버/구글 봇이 페이지 내용을 수집할 때, 가끔 내용이 다 완성되기 전에 수집해버려서 "빈 페이지"로 인식할 위험이 있습니다.
- **해결책**:
    - 1단계(현재 완료): `React Helmet`으로 페이지 접속 시 브라우저 제목과 설명을 강제로 바꿔주었습니다. 구글은 이걸 잘 읽습니다.
    - 2단계(추천): 네이버 검색 노출이 매우 중요하다면 추후 **SSR(Server Side Rendering)** 도입을 고려해야 합니다. (개발 비용/시간이 많이 듭니다)

### 2) 콘텐츠 운용 시 주의사항 (운영자가 할 일)
- **상품명**: 검색 사용자가 실제로 검색할법한 키워드를 포함하세요.
    - ❌ 나쁜 예: `Premium Speaker A-100` (아무도 안 검색함)
    - ⭕ 좋은 예: `야외 행사용 대형 스피커 JBL A-100 렌탈` (핵심 키워드 포함)
- **상품 설명**: 첫 2줄이 가장 중요합니다. 구글 검색 결과 미리보기에 뜹니다.

### 3) 등록해야 할 것들 (지금 바로 하세요!)
개발자가 코드를 넣었어도, 검색엔진 주인에게 "우리 사이트 여기 있어"라고 신고해야 합니다.
1. **네이버 서치어드바이저** 접속 (https://searchadvisor.naver.com/)
    - 사이트 등록 후 `요청 > 사이트맵 제출` 메뉴에서 `https://human-partner.web.app/sitemap.xml` 제출
2. **구글 서치 콘솔** 접속 (https://search.google.com/search-console)
    - 동일하게 사이트맵 제출

## ❓ 3. 향후 질문/논의 포인트
1. **주요 타겟 키워드가 무엇인가요?** ("행사 장비", "노트북 렌탈" 등 주력 키워드를 알려주시면 메인 페이지 키워드를 수정해드립니다.)
2. **블로그/매거진 기능?**: 상품 페이지만으로는 검색 유입에 한계가 있습니다. 행사 꿀팁 등을 올리는 블로그 기능이 있으면 SEO에 아주 큰 도움이 됩니다.

## 📝 4. 작업 로그 (기술적 상세)
- `npm install react-helmet-async` 설치
- `index.tsx`: `HelmetProvider` 래핑
- `MainPage.tsx`: 메인 키워드 "종합 행사 장비 렌탈 플랫폼" 설정
- `ProductDetail.tsx`: 상품명 기반 동적 타이틀 및 OpenGraph(카톡 공유 미리보기) 설정
- `public/robots.txt`: Disallow `/admin/`, `/mypage/` 설정
